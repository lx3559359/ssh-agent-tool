# ShellPilot UI、AI 与配置安全韧性实施计划

1. 为 AI 历史分页、请求策略、分包恢复和凭据保护先补失败测试。
2. 实现 AI 历史渐进加载、请求超时/取消/有限重试和上下文上限回归保护。
3. 实现可恢复按需模块边界，并接入 AI、配置、状态、帮助和更新模块。
4. 实现 API Key 单独安全存储迁移、无密钥导入导出和模型健康详情。
5. 增加 1366/1920、125%/150%、双主题、长文本和中文输入法的 Electron 回归用例及截图基准。
6. 分析构建产物体积，确认主包变化且不拆终端核心。
7. 执行单元测试、E2E 回归、Lint、生产构建和 Windows 打包烟测。
