# ShellPilot v0.4.10

## [新增]

- 增加发布资产版本一致性门禁，在上传 GitHub、同步 ModelScope 和客户端在线更新前校验源码版本、安装包、便携包、更新清单、校验和与发布标签。
- 增加本地 Telnet、SSH 和 SFTP 稳定测试环境，减少外部网络与随机测试密钥对回归结果的干扰。

## [修复]

- 修复 SFTP 文件列表刷新后选中项失效的问题，文件复制、粘贴、重命名和右键操作可以继续使用正确的文件上下文。
- 修复部分旧版端到端测试依赖过时界面文案、选择器和外部 Telnet 服务而产生的误报。
- 修复测试 SSH 服务偶发生成无效密钥导致连接回归失败的问题。

## [改动]

- 现代化 24 项旧版端到端流程，覆盖工作区、网页与 Telnet 连接、AI 配置、快捷命令、更新中心以及本地和远程文件操作。
- 更新 GitHub 与 ModelScope 发布流程，发布前必须通过版本基线和本地资产完整性检查。
- 强化 Windows 客户端发布自检，保持 SSH 终端核心不变，降低质量改动对日常连接功能的影响。
