# ShellPilot v0.4.5

## [新增]

- 新增每个 SSH 会话独立的“AI 接管”开关，默认关闭；只有用户主动开启后，Agent 才能在该会话内执行受控操作，关闭会话或断线会立即撤销授权。
- 新增分级自治执行链路。只读操作可按策略执行，修改、删除、上传、服务变更和本机命令等风险操作会弹出二次确认，并展示目标、命令、影响范围、风险、恢复与验证方式。
- 新增本地用户 Skill 包，默认不内置任何 Skill。支持通过对话生成草稿，也支持手动导入、编辑、校验、启用、禁用、回滚和删除。
- 新增结构化 SSH 运维工具、会话资源隔离、执行证据、任务取消与审计保留能力。

## [修复]

- 修复多个 SSH 标签页同时使用 Agent 时被全局运行锁相互阻塞的问题，现按会话隔离任务与授权。
- 修复 Agent 取消后后台请求、远程命令或文件传输可能晚到并继续更新状态的问题。
- 修复旧版 AI 配置中的 Skill 迁移可能产生不稳定标识或被意外启用的问题；迁移结果始终为待审查草稿。
- 修复 Skill IPC 在质量日志隔离环境中提前解析本地目录导致模块加载失败的问题。

## [改动]

- 所有 Agent 工具统一经过策略网关，不允许 Skill 或模型绕过权限、确认、会话边界和执行后验证。
- Agent 输出、日志与证据采用有界存储和脱敏处理；Skill 文件限制在应用数据目录，不允许任意根目录访问。
- AI 设置页保留现有布局，在高级设置中增加 Skill 管理入口；SSH 右侧 AI 面板增加当前会话接管控制，不新增主导航栏目。
- 新增 AI 接管与 Skill 管理端到端测试、性能/压力测试及 `smoke:ai-takeover` 发布前冒烟命令。
