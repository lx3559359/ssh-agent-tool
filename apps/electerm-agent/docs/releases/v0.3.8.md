# ShellPilot v0.3.8

## [新增]

- 新增面向初学者的完整帮助中心，覆盖首次连接、认证、SSH、SFTP、AI、Agent、快捷命令、端口转发、MCP、CLI、备份迁移、在线更新和故障排查。
- 新增详细离线中文使用手册，并补充大日志、压缩日志、附件分析、安全回滚和常见问题说明。

## [修复]

- 修复 AI 对话框底部与客户端顶部重复显示“帮助”入口的问题。
- 修复工作区内部与客户端标题栏重复显示最小化、最大化和关闭按钮的问题。
- 修复模型 API 配置页仍引用 Electerm 外部 Wiki 地址的问题。

## [改动]

- 帮助入口统一保留在客户端顶部，模型 API 页面改为 ShellPilot 内置简短配置步骤。
- 窗口控制统一由客户端最外层标题栏负责，工作区保留布局、全屏和搜索等终端工具按钮。
- 后续正式发布将同步检查客户端帮助中心、离线文档和结构化更新说明。
