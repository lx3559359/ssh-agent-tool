# ShellPilot v0.4.7

## [新增]

- 新增 0.4.7 在线更新发布，用新的版本号承接 0.4.6 的完整 AI 接管只读快路径、工具卡和风险确认体验，确保已经安装 0.4.6 的用户也能通过检查更新获取最新构建。

## [修复]

- 修复 0.4.6 同版本补发无法被已安装 0.4.6 客户端识别为更新的问题；本版本以 0.4.7 重新发布相同稳定能力，使在线更新比较逻辑可以正确发现新版本。

## [改动]

- 更新 Windows 发布资产、GitHub Releases 和 ModelScope 国内更新源的版本元数据到 0.4.7；安装包、便携包、latest.yml、checksums 和更新 approval manifest 均随新版本重新生成。
