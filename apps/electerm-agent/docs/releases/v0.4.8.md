# ShellPilot v0.4.8

## [新增]

- 新增多套 UI 配色方案与实时预览，客户端外观可独立调整，不改变 SSH 终端配色。
- 新增 20 种 UI 字体预设，支持搜索、可用性检测、即时预览、应用与取消；不影响 SSH 终端字体。
- 新增简体中文与英文的即时预览与切换，并补齐设置、快捷命令、服务与容器识别等次级界面文案。

## [修复]

- 固定所有 UI 主题下的 SSH 终端背景为近黑色，并自动修复低对比度前景色，避免终端文字难以辨认。
- 修复英文界面和窄屏、缩放场景下的顶栏溢出与文字挤压问题。
- 修复自动更新提示覆盖 AI 输入和其他正在进行的操作的问题。

## [改动]

- 统一美化设置与各配置页、弹窗、右键菜单、表单和快捷命令面板，加入节制的层次、边界与阴影。
- 保留原有 UI 主题编辑与导入能力，同时将 UI 主题、终端主题、UI 字体和界面语言解耦，保持 SSH、SFTP、AI 与更新流程的原有功能。
