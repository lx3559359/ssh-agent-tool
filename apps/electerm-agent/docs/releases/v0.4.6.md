# ShellPilot v0.4.6

## [新增]

- 新增 AI 接管的只读 exec 快路径。静态、受支持的 SSH 查询命令通过独立且有界的 exec 通道执行，不写入交互终端，也不要求二次确认。
- 新增可审计的 Agent 工具卡，展示命令、目标、耗时、退出码、截断状态和脱敏输出，并支持复制命令与安全地填入当前终端。
- 新增 Agent 运行图标；任务执行期间发送位置显示旋转状态，完成、失败或取消后恢复发送图标。

## [修复]

- 修复人工输入 SSH 命令也触发二次确认的问题；人工命令直通当前终端，AI 发起的风险操作仍由安全事务保护。
- 修复风险命令可能出现重复确认的问题；现在每个具体风险事务只显示一次详细确认，即风险单弹窗。
- 修复缺少 Shell Integration 时只读查询等待过久或无法可靠下发的问题，并保持取消、超时、端点变化和输出边界保护。

## [改动]

- 只读、写入和控制类 Agent 工具在执行前重新分类；只读操作零确认，修改和控制操作继续要求一次带目的、影响范围与验证方式的详细确认。
- “填入终端”仅在同一 SSH 会话、可信普通提示符且当前输入为空时可用，只填入命令而不自动按 Enter。
- 完成真实 VPS 只读验证，固定只读命令均通过正常客户端连接与 Agent 工具链执行；验证过程不记录服务器地址、凭据或命令输出。
