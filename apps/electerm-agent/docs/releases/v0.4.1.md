# ShellPilot v0.4.1

## [新增]

- 新增 AI 诊断弹窗关闭状态的启动回归测试，覆盖未选择异常目标时的主界面加载场景。

## [修复]

- 修复升级到 `0.4.0` 后，AI 诊断弹窗在目标为空时读取 `target.data`，导致客户端启动直接进入“界面发生错误”的问题。
- 修复错误分包名称容易被误判为剪贴板数据损坏的问题；本次故障不需要清空服务器、SSH、SFTP、AI 或历史配置。

## [改动]

- 诊断目标名称改为统一的空值安全解析；关闭弹窗时使用“异常目标”占位，不影响正常异常服务、容器和平台诊断。
- 保持 SSH、SFTP、安全操作中心与本地数据库格式不变，可直接从 `0.4.0` 平滑升级。
