# ShellPilot v0.3.7

## [新增]

- 新增客户端内中文帮助中心和离线中文使用文档。
- 新增在线更新源选择：自动选择、ModelScope 国内源、GitHub。
- 新增 SFTP 一键备份、安全删除、恢复记录和安全操作中心。
- 新增更多表单式服务器维护快捷命令，以及修改类操作的回滚保护。
- AI 助手支持多个 API 配置、多模型切换、大文件分段读取、压缩日志和文件附件分析。

## [修复]

- 修复仅 SFTP 会话执行备份时提示 `do not support cp operation in sftp mode` 的问题。
- 修复 SFTP 文件夹和二进制文件无法安全递归备份的问题。
- 修复部分 AI 对话、附件、快捷命令和界面布局的稳定性问题。

## [改动]

- 更新中心和设置页现在都能选择并记住更新源。
- 自动更新继续采用人工发布审批机制，只有确认发布的版本才会推送给客户端。
- 更新日志统一使用 `[新增]`、`[修复]`、`[改动]` 分类展示。
