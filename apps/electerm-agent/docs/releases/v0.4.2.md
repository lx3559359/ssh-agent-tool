# ShellPilot v0.4.2

## [新增]

- 新增设置中心搜索、简体中文与英文即时预览，以及适配浅色和深色界面的多套 UI 配色方案。
- 新增主题卡片、实时预览和主题字段双语说明，便于在应用前确认界面效果。
- 新增设置页、右键菜单、SFTP、快捷命令、组件和 AI 配置区域的双语界面覆盖。

## [修复]

- 修复浅色 UI 主题下终端文字与固定黑色背景对比度过低、难以辨认的问题。
- 修复主题删除、设置搜索、右键菜单定位与焦点切换等边界场景。
- 修复部分窄窗口和长文本场景中的挤压、溢出与菜单可读性问题。

## [改动]

- 统一设置页、配置表单和右键菜单的视觉层级、间距、圆角与交互反馈。
- UI 主题与终端主题保持解耦；无论界面配色如何切换，SSH 终端背景始终保持近黑色，并自动保证文字可读性。
- 保持 SSH、SFTP、AI、快捷命令、配置存储和连接流程不变，可直接从 `0.4.1` 平滑升级。
