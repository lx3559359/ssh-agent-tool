# ShellPilot v0.4.9

## [新增]

- 扩充服务器维护与故障排查快捷命令，覆盖系统、存储、网络安全、容器、服务、计划任务、主机名、时区、Swap、Hosts 和防火墙等常见场景。
- 为涉及修改的快捷命令补齐参数校验、执行前检查、恢复点、审计记录和安全中心快捷回滚能力。
- 增加快捷命令风险筛选与维护命令恢复注册，便于在多条命令中区分只读检查和可回滚修改。

## [修复]

- 修复客户端启动阶段当前标签为空时快捷命令上下文读取失败，避免界面白屏或启动异常。
- 修复 Shell Integration 检测后误丢弃 OSC 生命周期数据的问题，恢复快捷命令退出码跟踪和安全中心任务及时完成。
- 修复 Bash 已配置 `PROMPT_COMMAND` 时命令跟踪状态被标题更新等提示命令干扰的问题。
- 修复只读快捷命令在 Shell Integration 暂不可用时无法执行的问题，同时保持修改类命令严格禁止降级执行。
- 修复服务、防火墙、网络、时区、Swap、Hosts、主机名和计划任务等恢复流程中的竞态、状态漂移和撤销边界问题。

## [改动]

- 合并 UI 美化与中文界面改进，统一设置、配置页、弹窗、右键菜单、表单和快捷命令面板的层级与可读性。
- 只读快捷命令采用受限降级策略；仅允许无需确认的前台只读操作直通，Agent、后台任务和修改操作仍要求可靠跟踪。
- 强化真实 SSH 会话的连续命令审计，并增加隔离 SFTP 创建、读写、重命名、恢复和清理回归测试。
